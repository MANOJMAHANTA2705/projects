🎓 Student Database System (VB)
📌 Overview

The Student Database System is a desktop application developed using Visual Basic (VB) to efficiently manage student records. It allows users to store, update, search, and delete student information in a structured way.

🚀 Features
➕ Add new student records
🔍 Search student details
✏️ Update existing records
❌ Delete student records
📋 Display all student data
🖥️ User-friendly GUI interface
🛠️ Technologies Used
Programming Language: Visual Basic (VB.NET / VB6)
Database: MS Access / SQL Server
IDE: Visual Studio
📂 Project Structure
Student-Database-System/
│── Form1.vb
│── Database.accdb / .mdf
│── Modules/
│── README.md
│── Other project files
⚙️ How to Run
Open the project in Visual Studio
Ensure database connection is properly configured
Press F5 or click Start to run the application
🗄️ Database Details
Stores student information like:
Name
Roll Number
Course
Contact Details
Connected using VB database connectivity (ADO.NET)
